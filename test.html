This HTML template contains a JavaScript function displayParagraph() that retrieves the extracted paragraph from the Flask template and sets it as the text content of the
element with the id "extracted-paragraph". The JavaScript function is called automatically when the page loads (onload="displayParagraph()"). Replace 'path/to/your/file.txt' with the actual file path of the file you want to process, and 'Cael' with your specified variable. After running the code, Flask will serve the web application, and you can access it in your browser. The extracted paragraph will be displayed within the
element on the web page.
